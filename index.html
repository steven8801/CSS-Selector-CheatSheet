CSS : Selector CheatSheet

By steven8801 (https://github.com/steven8801)

Type CSS Description
Type Selector p {

}		 CSS type selectors are used to match all elements of a given type or tag name.
Class Selector .class_name {

}		 CSS class selector matches elements based on the contents of their class attribute.
ID Selector #id_name {

}		 CSS ID selector matches elements based on the contents of their id attribute.
Descendant selector tbody tr td {

}		 CSS descendant selector combinator is used to match elements that are descended from another matched selector.
Chaining Selectors typeName.className {

}		 CSS selectors can be chained so that rule sets apply only to elements that match all criteria.
